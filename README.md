# Data-Science-With-Orange

## How to install

- Start a new virtual environment
``` bash
$ virtualenv venv -p python3
$ source venv/bin/bash/activate
```

- Install dependencies
``` bash
$ pip install orange3
$ pip install pyQt5
```

or 

``` bash
$ pip install -r requirements.txt
```

## How to start Orange
``` bash
$ orange-canvas
```
